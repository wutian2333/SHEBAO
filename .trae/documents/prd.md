
# 社保测算网页 - 产品需求文档

## 1. 项目概述

### 1.1 项目背景
开发一个简洁实用的社保测算网页，帮助用户快速计算个人社保缴纳金额，了解社保相关政策。

### 1.2 目标用户
- 需要了解社保缴纳情况的在职人员
- 准备入职新工作需要了解社保待遇的求职者
- 对社保政策感兴趣的普通用户

### 1.3 核心功能
- 社保缴费基数输入
- 社保缴纳比例设置
- 实时计算社保费用
- 显示详细的社保构成说明

## 3. 功能需求

### 3.1 KV头图区域
- 展示品牌标语和主题
- 背景图采用渐变或简约设计
- 响应式适配不同屏幕

### 3.2 操作区域

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 缴费基数输入 | 用户输入月工资基数 | 高 |
| 城市选择 | 选择所在城市，影响缴费比例 | 高 |
| 缴费比例设置 | 预设常见比例，支持自定义 | 中 |
| 计算按钮 | 触发计算逻辑 | 高 |
| 结果展示 | 显示个人/单位/总计金额 | 高 |

### 3.3 说明区域

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 社保构成说明 | 解释养老保险、医疗保险等构成 | 中 |
| 计算公式 | 展示计算方法 | 中 |
| 常见问题 | FAQ解答 | 低 |

---

## 4. 设计规范

### 4.1 设计风格
- **风格**: 简洁现代、专业可信
- **色调**: 蓝色为主色调（#1E88E5），配合灰色系
- **字体**: 思源黑体 / Noto Sans SC

### 4.2 响应式设计
- 桌面端: 完整布局
- 移动端: 单列布局，适配小屏幕

## 5. 技术方案

### 5.1 技术栈
- **框架**: React + TypeScript
